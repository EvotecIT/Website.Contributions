# Posts

Create one folder per article:

```text
posts/<language>/<year>/<slug>/index.md
posts/<language>/<year>/<slug>/cover.jpg
posts/<language>/<year>/<slug>/images/screenshot-01.png
```

Examples:

```text
posts/en/2026/building-dhcp-reports/index.md
posts/pl/2026/raporty-dhcp-w-powershell/index.md
```

Keep every image for a post inside that post folder.
