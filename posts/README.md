# Posts

Create one folder per article:

```text
posts/<language>/<slug>/index.md
posts/<language>/<slug>/cover.webp
posts/<language>/<slug>/images/screenshot-01.webp
```

Examples:

```text
posts/en/building-dhcp-reports/index.md
posts/pl/raporty-dhcp-w-powershell/index.md
```

Keep every image for a post inside that post folder.

Blog contribution folders currently use `en` for English and `pl` for Polish. Accepted local image formats are PNG, JPG, JPEG, WEBP, and GIF.
